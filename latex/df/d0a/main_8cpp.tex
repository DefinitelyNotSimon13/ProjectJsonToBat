\begin{DoxyNote}{Note}
Example of using the logging wrapper! For some cases, I thought would be most common, there are definitions/macros (OUTPUT, LOG\+\_\+\+INFO, LOG\+\_\+\+WARNING, LOG\+\_\+\+ERROR, LOG\+\_\+\+DEBUG) that can be used. For other cases, the utils\+::log function can be used directly. Possibilitys\+:
\begin{DoxyItemize}
\item utils\+::log(bool verbose = false) -\/ Logs INFO, verbose can be set to true
\item utils\+::log(\+Log\+Level level) -\/ Logs the given level, verbose is false, however only INFO can be verbose anyway, as all other always print to the console
\item utils\+::log(comst std\+::string \&prefix, bool verbose = false) -\/ Logs INFO, with a custom prefix, verbose can be set to true
\item utils\+::log(const std\+::string \&prefix, Log\+Level level, bool verbose = false) -\/ Logs the given level, with a custom prefix, verbose can be set to true
\end{DoxyItemize}
\end{DoxyNote}
All of the above return a std\+::ostream, so you can use it like e.\+g. a normal std\+::cout!
